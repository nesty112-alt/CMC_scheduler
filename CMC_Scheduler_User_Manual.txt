======================================================================
                  CMC 스케줄러 (CMC Multi-Scheduler) 사용자 설명서
======================================================================

본 프로그램은 사용자가 지정한 파일(파이썬 스크립트, 실행 파일)이나 
엑셀 매크로(VBA)를 원하는 요일, 날짜, 시간에 맞춰 자동으로 실행해 주는 
똑똑한 '자동화 스케줄러' 프로그램입니다.

컴퓨터를 켜두기만 하면 정해진 시간에 알아서 업무를 대신 처리해 줍니다.

----------------------------------------------------------------------
1. 프로그램의 핵심 특징 💡
----------------------------------------------------------------------
▶ 다양한 파일 지원
   - 파이썬 파일(.py), 일반 프로그램(.exe)은 물론, 엑셀 매크로 파일(.xlsm, .xlsb)까지 
     원하는 대로 등록하여 자동으로 실행할 수 있습니다.
▶ 화면을 닫아도 계속 작동 (백그라운드 실행)
   - 프로그램 우측 상단의 [X] 버튼을 눌러 창을 닫아도 프로그램이 완전히 꺼지지 않고, 
     윈도우 작업 표시줄 우측 하단(시계 옆 숨겨진 아이콘 영역)에서 계속 작동합니다.
▶ 컴퓨터를 켜면 자동으로 시작
   - 'PC 부팅 시 자동 시작' 기능을 켜두면, 컴퓨터를 켤 때마다 프로그램이 
     알아서 켜져 백그라운드에서 스케줄을 대기합니다.
▶ 안전한 자동 저장
   - 등록해 둔 스케줄 목록은 프로그램이 꺼지거나 컴퓨터를 재부팅해도 
     안전하게 파일로 저장되어 그대로 유지됩니다.

----------------------------------------------------------------------
2. 초보자를 위한 스케줄 등록 5단계 가이드 🚀
----------------------------------------------------------------------
[1단계] 실행할 파일 선택하기
  - 화면 상단의 [파일 찾아보기] 버튼을 누릅니다.
  - 자동으로 실행하고 싶은 파이썬 파일(.py), 프로그램(.exe), 또는 엑셀 매크로 파일(.xlsm, .xlsb)을 선택합니다.

[2단계] 엑셀 매크로 이름 입력하기 (★엑셀 파일 선택 시에만 필수!)
  - 선택한 파일이 엑셀 매크로 파일인 경우, 'VBA 매크로명' 입력창이 활성화됩니다.
  - 실행할 매크로의 정확한 이름을 입력해 주세요. (예: Module1.MyMacro)
  - ※ 일반 파이썬이나 프로그램 파일은 입력하지 않고 그냥 넘어가시면 됩니다.

[3단계] 실행 기간 설정하기
  - 언제부터 이 스케줄을 시작할지 '시작일'을 지정합니다.
  - 특정 날짜까지만 실행하고 중단하고 싶다면 '종료일 지정' 체크박스를 체크한 후 날짜를 선택합니다.
  - 체크를 해제해 두면 종료일 없이 계속(무기한) 실행됩니다.

[4단계] 반복 방식 및 시간 주기 설정하기
  - 매주 반복: 실행하고 싶은 요일(월~일)을 체크합니다. (여러 요일 중복 선택 가능)
  - 매월 반복: 매달 몇 일에 실행할지 선택합니다. (예: 매월 10일 / 매월 말일 선택 가능)
  - 실행 시간: 오른쪽 '실행 시간' 입력창에서 몇 시 몇 분에 실행할지 정확히 설정합니다.

[5단계] 등록 완료하기
  - 모든 설정을 마쳤다면 [스케줄 추가] 버튼을 누릅니다.
  - 아래쪽 '스케줄 목록' 테이블에 내가 등록한 작업이 추가된 것을 확인할 수 있습니다.

----------------------------------------------------------------------
3. 화면 메뉴 및 기능 설명 🖥️
----------------------------------------------------------------------
■ PC 부팅 시 백그라운드로 자동 시작 (체크박스)
  - 이 옵션을 체크해 두면 컴퓨터 전원을 켤 때 프로그램이 자동으로 실행됩니다. 
  - 이때 화면이 크게 뜨지 않고, 작업 표시줄 시계 옆으로 조용히 켜지므로 편리합니다.

■ 스케줄 목록 테이블 (중앙 화면)
  - 작업 대상: 내가 등록한 파일명과 매크로 이름이 표시됩니다. (마우스를 올리면 전체 경로 확인 가능)
  - 상태: 현재 작업이 '대기 중'인지, '실행 중'인지, 아니면 '정상 종료'되었거나 '오류 발생'했는지 실시간으로 보여줍니다.
  - 마지막 실행 / 다음 실행: 마지막으로 작동한 시간과 앞으로 작동할 예정 시간을 안내합니다.
  - 관리 [삭제] 버튼: 더 이상 실행하고 싶지 않은 스케줄은 [삭제] 버튼을 누르면 목록에서 지워지고 스케줄이 취소됩니다.

■ 실시간 실행 로그 (하단 화면)
  - 프로그램이 몇 시 몇 분에 어떤 작업을 시작했고 성공했는지, 혹은 왜 실패했는지 상세한 기록을 실시간으로 보여주는 창입니다.

----------------------------------------------------------------------
4. 창이 닫혔을 때! 당황하지 마세요 (트레이 아이콘 안내) 🔄
----------------------------------------------------------------------
- 프로그램의 [X] 버튼을 누르면 창이 완전히 꺼지는 것이 아니라 '숨김 상태'가 됩니다.
- 다시 화면을 열고 싶다면?
  1. 윈도우 우측 하단 시계 옆의 [ ∧ ] 모양 버튼(숨겨진 아이콘 표시)을 누릅니다.
  2. 컴퓨터 모양의 [CMC 스케줄러 아이콘]을 찾습니다.
  3. 아이콘을 '더블 클릭'하거나, '마우스 우클릭 -> [관리자 화면 열기]'를 선택하면 창이 다시 나타납니다.
- 프로그램을 완전히 종료하고 싶다면?
  - 스케줄러를 완전히 끄려면 반드시 트레이 아이콘을 '마우스 우클릭'한 후 **[완전히 종료]**를 클릭해야 합니다.

----------------------------------------------------------------------
5. 사용 시 반드시 지켜야 할 주의사항 ⚠️
----------------------------------------------------------------------
① 엑셀 매크로 실행 시 주의점
   - 스케줄러가 엑셀 매크로를 실행할 때는 화면에 엑셀 창이 뜨지 않고 백그라운드에서 조용히 실행된 후 자동 저장됩니다.
   - 만약 매크로 코드 안에 "확인" 버튼을 눌러야만 넘어가는 알림창(MsgBox 등)이 들어있다면, 화면에 보이지 않아 누를 수 없으므로 프로그램이 멈춰버릴 수 있습니다. 매크로 내부의 알림창 코드는 미리 제거해 주세요.
② 파일 위치 및 이름 변경 금지
   - 스케줄러는 등록 당시의 파일 경로를 기억하여 실행합니다. 등록 후에 원본 파일의 위치를 옮기거나 파일 이름을 바꾸면 "파일이 존재하지 않습니다"라는 에러가 발생합니다. 경로가 바뀌었다면 기존 스케줄을 삭제하고 다시 등록해 주세요.
③ 파이썬 라이브러리 환경 관련
   - 파이썬 파일(.py)을 등록해 두었을 경우, 해당 스크립트가 실행되는 데 필요한 외부 라이브러리(Pandas 등)가 컴퓨터에 올바르게 설치되어 있어야 정상적으로 작동합니다.
